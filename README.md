# ResqNet Landing Page

ResqNet is a responsive landing page built using **HTML and CSS** that promotes an emergency power solution service for blackout situations. The website includes a hero section, product features, customer reviews carousel, subscription plans, and a footer with contact information.

The layout uses **Flexbox and CSS Grid** for structure and **media queries** for responsiveness across different screen sizes. A **CSS-only testimonial slider** is implemented using radio buttons and the `:checked` pseudo-class, allowing users to switch between reviews without JavaScript.

This project demonstrates fundamental front-end development concepts such as responsive design, layout systems, and interactive UI elements using pure HTML and CSS.
